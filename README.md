# Multi-Store Sales Forecasting using Time Series Regression & XGBoost

## Overview

Developed a sales forecasting pipeline on historical retail sales data to predict product demand across multiple stores.

## Project Objectives

- Forecast future sales using historical data
- Capture seasonality and demand trends
- Compare baseline and advanced machine learning models
- Improve forecasting accuracy through feature engineering

## Techniques Used

- Time Series Analysis
- Feature Engineering
- Time Series Cross Validation
- Linear Regression
- XGBoost
- Hyperparameter Tuning
- Feature Importance Analysis

## Feature Engineering

Created:

- Lag Features
- Rolling Window Features
- Date Based Features
- Store-Level Features

to capture sales trends, seasonality and demand patterns.

## Results

| Model | Validation RMSE |
|---------|---------|
| Linear Regression | 1.05 |
| XGBoost | 0.93 |

Achieved approximately 11% improvement in forecasting accuracy over the baseline model.
